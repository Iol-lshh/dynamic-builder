# Infra Level

물리/가상 인프라 수준에서 사고한다. 서버, 네트워크, 스토리지 구성과 리소스 의존성에 집중한다.

## 렌즈

- **출발점**: 물리/가상 인프라 구성 요소와 그 관계
- **기술 수준**: "어떻게 구성되어 있는가(how)" — 서버, 네트워크, 스토리지 배치
- **코드 참조**: IaC 코드, 배포 스크립트, 환경 설정 확인 용도
- **산출물 수준**: 인프라 구성도, 리소스 의존성, 병목 지점, 가용성 분석

## 이 관점이 적합한 작업

- 인프라 병목 및 단일 장애점(SPOF) 분석
- 리소스 프로비저닝 전략 수립
- 네트워크 토폴로지 검토
- 고가용성(HA) 및 재해 복구(DR) 설계 검토
