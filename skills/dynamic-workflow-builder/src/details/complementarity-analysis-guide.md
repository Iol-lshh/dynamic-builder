# Complementarity Analysis Guide

에세이 간 질적 보완성을 분석하는 기준을 정의한다.

## 분석 대상

- 입력: extract-scores.md (6차원 스코어 매트릭스)
- 참조: 원본 essay-synthesis.md 파일들 (종합 소견, 관점별 분석 근거)

원본 essay-synthesis.md 파일은 extract-scores.md에 기록된 파일 경로를 통해 접근한다. 스코어만으로는 파악할 수 없는 질적 특성(논조, 강점/약점의 구체적 내용, 관점의 독특성)을 원본에서 확인한다.

## 보완성 평가 기준

### 1. 관점 다양성 (Perspective Diversity)

- 에세이들이 동일 주제를 서로 다른 각도에서 다루는가
- 한 에세이가 놓친 시각을 다른 에세이가 보완하는가
- 평가: 높음 / 중간 / 낮음

### 2. 논조 보완성 (Tone Complementarity)

- 낙관적 에세이와 비관적 에세이가 균형을 이루는가
- 구조 수용적 에세이와 해체적 에세이가 공존하는가
- 스코어상 낙관도/해체도가 유사해도 논조의 구체적 방향이 다를 수 있음에 주의
- 평가: 높음 / 중간 / 낮음

### 3. 강점/약점 상호 보완 (Strength-Weakness Reciprocity)

- 한 에세이의 약점 영역(낮은 척도)을 다른 에세이의 강점이 보완하는가
- 예: 에세이 A가 문제인식은 높지만 대안해상도가 낮고, 에세이 B가 그 반대인 경우 -> 높은 보완성
- 평가: 높음 / 중간 / 낮음

### 4. 내용 중복도 (Content Overlap)

- 에세이들이 동일한 논점을 반복하는 정도
- 중복이 높으면 팀으로 묶었을 때 다양성이 떨어짐
- 평가: 낮음(좋음) / 중간 / 높음(나쁨)

## 산출물 형식

에세이 쌍별 보완성 매트릭스를 작성한다.

### 쌍별 평가 테이블

| 에세이 A | 에세이 B | 관점 다양성 | 논조 보완성 | 강점/약점 보완 | 내용 중복도 | 종합 보완성 | 근거 요약 |
|----------|----------|-------------|-------------|----------------|-------------|-------------|-----------|

### 종합 보완성 산출

- 4개 기준의 정성 평가를 종합하여 쌍별 종합 보완성을 "높음/중간/낮음"으로 판정한다.
- 종합 보완성 판정의 근거를 1-2문장으로 기술한다.

## 주의사항

- 스코어 유사성과 질적 보완성은 별개이다. 스코어가 비슷한 에세이도 관점이 다르면 보완성이 높을 수 있고, 스코어가 다른 에세이도 같은 관점이면 보완성이 낮을 수 있다.
- 보완성 판단은 반드시 원본 텍스트의 구체적 근거에 기반해야 한다. 스코어 차이만으로 보완성을 추론하지 않는다.
