# Grouping Output Format

에세이 그룹화 워크플로우의 optimize-grouping 스텝 산출물 형식을 정의한다.

## 구조

산출물은 두 배정안을 포함한다.

---

## 배정안 1: 부분 최적화 (Local Optimization)

### 요약

| 항목 | 값 |
|------|-----|
| 전략 | 단일 최강 팀 극대화 |
| 최고 팀 총점 | {점수}/600 |
| 탐색 방법 | 전수 탐색 / 휴리스틱 (해당 시 명시) |

### 팀별 배정

#### 팀 1 (최고 보완 조합)

| 에세이 ID | 제목 | 낙관도 | 해체도 | 문제인식 | 대안해상도 | 내용응집도 | 관점차별도 |
|-----------|------|--------|--------|----------|------------|------------|------------|

**팀 점수**: {6척도 팀점수 각각} / 팀총점: {합산}/600

**배정 근거 (정량)**: 팀 점수 계산 과정 요약
**배정 근거 (정성)**: 보완성 분석에서의 판단 근거

#### 팀 2 ~ 팀 N

(동일 형식 반복)

---

## 배정안 2: 전체 최적화 (Global Optimization, Maximin)

### 요약

| 항목 | 값 |
|------|-----|
| 전략 | 최약 팀 점수 극대화 (maximin) |
| 최약 팀 총점 | {점수}/600 |
| 탐색 방법 | 전수 탐색 / 휴리스틱 (해당 시 명시) |

### 팀별 배정

(배정안 1과 동일 형식)

---

## 두 배정안 비교

| 항목 | 부분 최적화 | 전체 최적화 |
|------|------------|------------|
| 최고 팀 총점 | | |
| 최약 팀 총점 | | |
| 팀 간 점수 편차 | | |
| 권장 상황 | 최강 팀이 필요한 경우 | 팀 간 균형이 중요한 경우 |
