# 에세이 분석 척도 정의

6개 척도를 100점 만점 스코어로 평가한다. 각 척도는 독립적으로 채점된다.

---

## 1. 낙관도 (Optimism)

텍스트가 대상이나 상황에 대해 긍정적 전망을 제시하는 정도.

| 점수 구간 | 설명 |
|-----------|------|
| 0-20 | 극단적 비관. 대안 없는 절망적 서술 |
| 21-40 | 비관적. 문제 중심 서술, 희망 요소 미약 |
| 41-60 | 중립. 긍정/부정 균형 또는 판단 유보 |
| 61-80 | 낙관적. 가능성과 대안을 적극 제시 |
| 81-100 | 극단적 낙관. 비판적 검토 없는 긍정 일변도 |

**주의**: 높은 점수가 반드시 좋은 글을 의미하지 않는다. 극단적 낙관(81-100)은 비판적 사고 결여를 나타낼 수 있다.

---

## 2. 해체도 (Deconstruction Level)

선험적 가정을 수용하는 정도(0) vs 기존 구조를 해체하는 정도(100).

| 점수 구간 | 설명 |
|-----------|------|
| 0-20 | 강한 선험. 기존 권위·전통·공리를 무비판적 수용 |
| 21-40 | 선험 우세. 기존 프레임 내에서 사고, 부분적 의문 |
| 41-60 | 균형. 기존 구조를 인정하되 한계를 지적 |
| 61-80 | 해체 우세. 이항 대립의 위계를 전복, 숨은 전제 폭로 |
| 81-100 | 강한 해체. 모든 안정적 의미 체계를 의심, 근본적 해체 |

---

## 3. 문제 인식 (Problem Awareness)

텍스트가 다루는 문제를 얼마나 정확하고 깊이 있게 인식하는가.

| 점수 구간 | 설명 |
|-----------|------|
| 0-20 | 문제 미인식. 현상만 나열, 문제로 정의하지 않음 |
| 21-40 | 피상적 인식. 표면적 증상만 언급, 구조적 원인 미탐색 |
| 41-60 | 부분적 인식. 일부 원인 파악, 연결 관계 불완전 |
| 61-80 | 구조적 인식. 원인-결과 체계 파악, 맥락적 이해 |
| 81-100 | 심층 인식. 역사적·구조적·다층적 원인 분석, 문제의 본질 관통 |

---

## 4. 대안 해상도 (Alternative Resolution)

텍스트가 제시하는 대안이 얼마나 구체적이고 실현 가능한가.

| 점수 구간 | 설명 |
|-----------|------|
| 0-20 | 대안 부재. 문제 제기만 존재 |
| 21-40 | 추상적 대안. "~해야 한다" 수준의 당위적 제안 |
| 41-60 | 방향 제시. 대안의 방향은 있으나 구체적 경로 불명확 |
| 61-80 | 구체적 대안. 실행 가능한 단계와 조건을 명시 |
| 81-100 | 정밀 대안. 예상 장애·반론까지 고려한 실행 가능 로드맵 |

---

## 5. 내용 응집도 (Content Cohesion)

텍스트 내부의 논리적 연결성과 구조적 일관성.

| 점수 구간 | 설명 |
|-----------|------|
| 0-20 | 파편적. 단락 간 연결 없음, 논지 불명확 |
| 21-40 | 느슨한 연결. 주제는 있으나 논리적 흐름 단절 |
| 41-60 | 부분 응집. 일부 구간 논리적, 전체적 구조 불균형 |
| 61-80 | 높은 응집. 일관된 논지, 단락 간 유기적 연결 |
| 81-100 | 완전 응집. 모든 요소가 핵심 논지를 향해 수렴, 군더더기 없음 |

---

## 6. 관점 차별도 (Perspective Differentiation)

텍스트의 관점이 얼마나 독창적이며 클리셰를 벗어나는가.

| 점수 구간 | 설명 |
|-----------|------|
| 0-20 | 완전 클리셰. 상투적 표현과 통념의 반복 |
| 21-40 | 클리셰 우세. 간헐적 독자적 관점, 대부분 기존 담론 추종 |
| 41-60 | 혼재. 독자적 시각과 관습적 서술이 공존 |
| 61-80 | 차별화. 고유한 프레임이나 시각으로 대상을 재조명 |
| 81-100 | 강한 차별화. 기존 담론을 근본적으로 재정의하는 독창적 관점 |

---

## 채점 원칙

1. 각 척도는 독립적으로 채점한다 (상호 영향 금지)
2. 점수는 5점 단위로 부여한다 (예: 45, 50, 55)
3. 각 점수에 대해 텍스트의 구체적 근거를 1-2개 인용한다
4. 총점은 6개 척도의 단순 평균으로 산출한다
5. 총점 외에 각 척도별 점수를 반드시 개별 표시한다
