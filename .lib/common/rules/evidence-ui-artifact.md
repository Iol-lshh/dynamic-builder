# 증거 규격: UI 산출물

HTML 퍼블·Figma 노드·디자인 명세를 근거로 주장할 때 제시해야 하는 증거 포맷.

## HTML / 퍼블리싱

### 필수 항목

- 파일 경로
- 셀렉터 (CSS selector 또는 XPath)
- 해당 영역 원문 HTML 블록

### 포맷

````
출처: {path}
셀렉터: {css-selector}

```html
{원문 블록 그대로}
```
````

## Figma

### 필수 항목

- Figma URL 또는 파일 키
- 노드 ID
- 프레임·컴포넌트 경로
- 조회한 속성 원문

### 포맷

```
출처: {figma-url}
노드 ID: {node-id}
경로: {frame > group > component}

속성:
{원문 그대로}
```

## 금지 사항

- "퍼블에 버튼이 있다" 등 요약만 제시
- 셀렉터·노드 ID 없이 "여기에 있다"
- 스크린샷 없이 시각 주장만 제시
- 속성 의역 또는 재구성
