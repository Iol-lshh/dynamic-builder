# 증거 규격: 코드 실행

코드 컴파일·테스트·런타임 검증을 주장할 때 제시해야 하는 증거 포맷.

## 필수 항목

- 실행 명령어 (전체 command line)
- 작업 디렉토리 (cwd)
- stdout 원문 (발췌·생략 금지)
- stderr 원문 (있을 경우)
- 종료 코드 (exit status)

## 포맷

```
$ {command}
(cwd: {path})

--- stdout ---
{원문 그대로}

--- stderr ---
{원문 그대로}

exit: {code}
```

## 금지 사항

- "빌드 성공했습니다" 등 자기 서술만 제시
- 로그 발췌 후 "... 이하 생략 ..." 처리
- 실행 없이 "정상 동작할 것으로 보임" 류의 추론
- 명령어만 제시하고 출력 누락
