# Grouping Extract Guide

복수 essay-synthesis.md 파일에서 6차원 스코어를 추출하여 구조화된 매트릭스로 변환하는 절차를 정의한다.

## 입력 파일 포맷

각 essay-synthesis.md 파일은 다음 구조를 포함한다.

- **에세이 ID**: 파일명 또는 frontmatter에서 추출
- **6차원 스코어**: 본문 내 채점 결과 섹션에 척도별 점수(0-100, 5점 단위)가 기재됨
- **총점**: 6개 척도의 단순 평균

### 추출 대상 필드

| # | 필드명 | 추출 위치 | 형식 |
|---|--------|-----------|------|
| 1 | 에세이 ID | 파일명 또는 제목 | 문자열 |
| 2 | 낙관도 (Optimism) | 채점 결과 테이블 | 정수 0-100 |
| 3 | 해체도 (Deconstruction) | 채점 결과 테이블 | 정수 0-100 |
| 4 | 문제 인식 (Problem Awareness) | 채점 결과 테이블 | 정수 0-100 |
| 5 | 대안 해상도 (Alternative Resolution) | 채점 결과 테이블 | 정수 0-100 |
| 6 | 내용 응집도 (Content Cohesion) | 채점 결과 테이블 | 정수 0-100 |
| 7 | 관점 차별도 (Perspective Differentiation) | 채점 결과 테이블 | 정수 0-100 |
| 8 | 총점 | 6척도 단순 평균 | 소수점 1자리 |

## 추출 절차

1. 작업 디렉토리에서 모든 essay-synthesis.md 파일을 열거한다.
2. 각 파일에서 채점 결과 섹션을 찾는다 (테이블 또는 key-value 형식).
3. 6개 척도의 점수를 파싱한다. 값이 누락되거나 범위(0-100)를 벗어나면 해당 에세이를 오류로 표시한다.
4. 총점이 기재되어 있으면 검증한다 (6척도 평균과 일치 여부). 미기재 시 직접 계산한다.
5. 결과를 출력 매트릭스 형식으로 정리한다.

|-----------|--------|--------|----------|------------|------------|------------|------|

| essay-B   | 30     | 75     | 80       | 45         | 60         | 85         | 62.5 |

### 요약 통계

| 척도 | 평균 | 표준편차 | 최솟값 | 최댓값 |
|------|------|----------|--------|--------|
| (6척도 각각) | | | | |

### 오류 목록

추출 실패한 에세이가 있으면 ID와 사유를 기재한다. 오류가 없으면 "없음"으로 기재한다.

## 주의사항

- scoring-scales.md의 척도 정의와 점수 구간을 참조하여 추출값의 타당성을 판단한다.
- 5점 단위가 아닌 점수(예: 67)가 발견되면 오류로 표시한다.
- 에세이 수(M)를 매트릭스 상단에 명시한다. 이 값은 후속 스텝(compute-team-scores)의 탐색 전략 선택에 사용된다.
