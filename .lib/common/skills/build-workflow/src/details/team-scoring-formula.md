# Team Scoring Formula

에세이 그룹 조합별 팀 점수 계산 공식을 정의한다.

## 개별 척도 팀 점수

에세이 집합 T = {e1, e2, ..., ek}가 한 팀을 구성할 때, 척도 s에 대한 팀 점수:

### 중간값 최적 척도 (s = 낙관도, 해체도)

```
팀평균(s) = (e1.s + e2.s + ... + ek.s) / k
팀점수(s) = max(0, 100 - |팀평균(s) - 50| x 2)
```

### 최대값 최적 척도 (s = 문제인식, 대안해상도, 내용응집도, 관점차별도)

```
팀평균(s) = (e1.s + e2.s + ... + ek.s) / k
팀점수(s) = 팀평균(s)
```

## 팀 총점

```
팀총점 = 팀점수(낙관도) + 팀점수(해체도) + 팀점수(문제인식) + 팀점수(대안해상도) + 팀점수(내용응집도) + 팀점수(관점차별도)
```

- 6척도 균등 가중치 (각 척도 최대 100, 팀총점 최대 600)

## 부분 최적화 점수

```
부분최적화점수 = max(팀총점(T1), 팀총점(T2), ..., 팀총점(TN))
```

가장 높은 단일 팀의 총점을 극대화하는 배정을 선택한다.

## 전체 최적화 점수 (Maximin)

```
전체최적화점수 = min(팀총점(T1), 팀총점(T2), ..., 팀총점(TN))
```

모든 팀 중 가장 낮은 팀총점을 최대화하는 배정을 선택한다.

## 탐색 전략

에세이 수(M)와 그룹 수(N)에 따라 조합 수가 폭발적으로 증가할 수 있다.

### M <= 8: 전수 탐색

- 가능한 모든 그룹 조합을 열거하고 각 조합의 팀총점을 계산한다.
- 부분 최적화와 전체 최적화 각각의 최적 배정을 산출한다.

### M > 8: 휴리스틱 전략

- **Greedy 구성**: 부분 최적화는 최고 팀부터 탐욕적으로 구성. 전체 최적화는 가장 약한 팀에 가장 도움이 되는 에세이를 순차 배정.
- **상위 K 후보**: 각 단계에서 상위 K(K=5)개 후보를 유지하며 탐색 (beam search).
- **결과 표기**: 휴리스틱 적용 시 "근사 최적(approximate)" 임을 산출물에 명시한다.
