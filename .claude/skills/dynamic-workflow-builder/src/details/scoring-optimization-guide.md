# Scoring Optimization Guide

에세이 그룹화 워크플로우에서 사용하는 척도별 최적값 정의와 두 최적화 모드를 기술한다.

## 척도별 최적값

| 척도 | 최적값 유형 | 팀 점수 계산 방식 | 비고 |
|------|------------|------------------|------|
| 낙관도(Optimism) | 중간값(50) | `100 - |팀평균 - 50| x 2` | 팀 평균이 50이면 만점(100), 0 또는 100이면 0점 |
| 해체도(Deconstruction) | 중간값(50) | `100 - |팀평균 - 50| x 2` | 위와 동일 |
| 문제 인식(Problem Awareness) | 최대값 | 팀평균 그대로 | 높을수록 좋음 (0-100) |
| 대안 해상도(Alternative Resolution) | 최대값 | 팀평균 그대로 | 높을수록 좋음 |
| 내용 응집도(Content Cohesion) | 최대값 | 팀평균 그대로 | 높을수록 좋음 |
| 관점 차별도(Perspective Differentiation) | 최대값 | 팀평균 그대로 | 높을수록 좋음 |

### 중간값 최적 척도 (낙관도, 해체도)

- 팀 내 에세이들의 해당 척도 점수 평균이 50에 가까울수록 좋다.
- 극단적으로 낙관적이기만 하거나 극단적으로 해체적이기만 한 팀은 낮은 점수를 받는다.
- 공식: `팀점수 = max(0, 100 - |팀평균 - 50| x 2)`

### 최대값 최적 척도 (문제인식, 대안해상도, 내용응집도, 관점차별도)

- 팀 내 에세이들의 해당 척도 점수 평균이 높을수록 좋다.
- 공식: `팀점수 = 팀평균` (0-100 범위 그대로)

## 두 최적화 모드

### 부분 최적화 (Local Optimization)

- 목표: 단일 팀의 팀 점수(6척도 합산)를 극대화한다.
- 방법: 가장 높은 팀 점수를 만들 수 있는 에세이 조합을 우선 선택하고, 나머지 에세이로 후속 팀을 배정한다.
- 결과: 최강 팀 1개가 나타나지만, 나머지 팀은 상대적으로 약할 수 있다.

### 전체 최적화 (Global Optimization, Maximin)

- 목표: 모든 팀의 팀 점수 중 최솟값을 최대화한다 (maximin 전략).
- 방법: 가장 약한 팀의 점수가 가장 높아지도록 에세이를 배분한다.
- 결과: 모든 팀이 고르게 강하지만, 최강 팀의 절대 점수는 부분 최적화보다 낮을 수 있다.
