# 에세이 분석 산출물 포맷

## 척도별 분석 산출물 (analyst)

각 관점(경험주의/선험주의/해체주의)의 분석가는 아래 포맷으로 산출물을 작성한다.

```markdown
# {척도명} 분석 — {관점}

## 관점 요약
{이 관점에서 해당 척도를 어떻게 바라보는지 1-2문장}

## 분석

### 관찰
{텍스트에서 발견한 구체적 요소들}

### 해석
{관찰 결과에 대한 이 관점에서의 해석}

### 스코어 제안
- **점수**: {0-100, 5점 단위}
- **근거**: {텍스트에서 인용한 구체적 근거 1-2개}

## 주요 발견
- {발견 1}
- {발견 2}
- ...
```

## 척도별 통합 산출물 (reconciler)

```markdown
# {척도명} — 다관점 통합 분석

## 관점별 요약
| 관점 | 스코어 제안 | 핵심 근거 |
|------|-----------|----------|
| 경험주의 | {점수} | {근거} |
| 선험주의 | {점수} | {근거} |
| 해체주의 | {점수} | {근거} |

## 합의점
{3개 관점이 동의하는 부분}

## 분기점
{관점 간 상충하는 판단과 각각의 근거}

## 통합 스코어
- **점수**: {0-100, 5점 단위}
- **산출 근거**: {어떤 관점의 판단을 어떻게 반영했는지}
```

## 최종 보고서 (synthesis)

```markdown
# 에세이 분석 보고서

## 텍스트 개요
{분석 대상 텍스트의 제목, 저자, 핵심 주제 요약}

## 척도별 분석 결과

| 척도 | 스코어 | 요약 |
|------|--------|------|
| 낙관도 | {점수} | {한 줄 요약} |
| 해체도 | {점수} | {한 줄 요약} |
| 문제 인식도 | {점수} | {한 줄 요약} |
| 대안 해상도 | {점수} | {한 줄 요약} |
| 내용 응집도 | {점수} | {한 줄 요약} |
| 관점 차별도 | {점수} | {한 줄 요약} |

## 총점
**{총점}/100** (6개 척도 평균)

## 상세 분석

### 1. 낙관도 ({점수}/100)
{통합 분석 내용}

### 2. 해체도 ({점수}/100)
{통합 분석 내용}

### 3. 문제 인식 ({점수}/100)
{통합 분석 내용}

### 4. 대안 해상도 ({점수}/100)
{통합 분석 내용}

### 5. 내용 응집도 ({점수}/100)
{통합 분석 내용}

### 6. 관점 차별도 ({점수}/100)
{통합 분석 내용}

## 종합 소견
{텍스트의 강점, 약점, 특이점에 대한 종합적 평가}
```
