# 논픽션 글 채점 기준 (nf-scoring-criteria)

## 목적

논픽션 글의 품질을 5개 항목으로 채점하여 100점 만점 점수를 산출한다.

---

## 채점 항목

### 1. 논증 타당성 (20점)

| 점수 | 기준 |
|------|------|
| 18~20 | 모든 주장에 구체적 근거가 뒷받침됨. 반론이 공정하게 다뤄짐. 논리적 비약 없음 |
| 14~17 | 대부분의 주장에 근거 존재. 일부 반론 처리 부족 또는 경미한 논리적 비약 |
| 10~13 | 근거가 일부 주장에만 존재하거나, 근거의 구체성이 부족. 반론 처리 미흡 |
| 0~9 | 근거 없는 주장이 다수이거나, 논리적 비약이 심각 |

### 2. 독자 적합성 (20점)

| 점수 | 기준 |
|------|------|
| 18~20 | 어휘·설명 깊이·톤이 대상 독자에 완벽히 맞춤. 독자의 전제 지식을 정확히 고려 |
| 14~17 | 대체로 독자에 맞으나, 일부 구간에서 어휘 수준 불일치 또는 불필요한 배경 설명 |
| 10~13 | 독자 수준과의 괴리가 다수 구간에서 발견됨. 전문 용어 과다 또는 과도한 단순화 |
| 0~9 | 독자를 고려하지 않은 글. 어휘·설명·톤이 대상 독자와 불일치 |

### 3. 구조 완성도 (20점)

| 점수 | 기준 |
|------|------|
| 18~20 | 도입-본문-결론 구조 명확. 소제목 간 논리적 전개. 분량 배분 적절. 도입이 독자를 끌어당김 |
| 14~17 | 구조가 존재하나, 일부 전환이 부자연스럽거나 분량 배분이 불균형 |
| 10~13 | 구조적 문제가 다수. 소제목 간 연결 부족 또는 도입/결론이 약함 |
| 0~9 | 구조가 불명확하거나 논리적 전개가 파악되지 않음 |

### 4. 문체 일관성 (20점)

| 점수 | 기준 |
|------|------|
| 18~20 | 글 전체에서 톤·문체·어조가 일관됨. 매체와 목적에 적합한 문체 유지 |
| 14~17 | 대체로 일관되나, 일부 구간에서 톤 변동 감지 |
| 10~13 | 톤 변동이 잦거나, 매체/목적에 부적합한 문체 혼용 |
| 0~9 | 문체가 일관되지 않아 읽기 불편. 톤이 산만함 |

### 5. 근거 활용도 (20점)

| 점수 | 기준 |
|------|------|
| 18~20 | nf-answer-analysis의 핵심 근거가 글 전반에 효과적으로 활용됨. 근거와 주장의 연결이 명확 |
| 14~17 | 근거가 대부분 활용되었으나, 일부 근거의 활용이 피상적 |
| 10~13 | 근거 활용이 부분적. 일부 핵심 근거 누락 또는 근거-주장 연결 약함 |
| 0~9 | 근거가 거의 활용되지 않거나, nf-answer-analysis 산출물과 괴리 |

---

## 총점 산출

| 항목 | 배점 |
|------|------|
| 논증 타당성 | 20 |
| 독자 적합성 | 20 |
| 구조 완성도 | 20 |
| 문체 일관성 | 20 |
| 근거 활용도 | 20 |
| **합계** | **100** |

---

## 산출물 형식

```markdown
## Score: {총점}/100

## 항목별 채점

| 항목 | 점수 | 근거 |
|------|------|------|
| 논증 타당성 (20) | {점수}/20 | {구체적 근거} |
| 독자 적합성 (20) | {점수}/20 | {구체적 근거} |
| 구조 완성도 (20) | {점수}/20 | {구체적 근거} |
| 문체 일관성 (20) | {점수}/20 | {구체적 근거} |
| 근거 활용도 (20) | {점수}/20 | {구체적 근거} |

## 개선 필요 사항

- {항목}: {개선 권고}
```

## retry 판정

- **score >= 80**: 합격. 다음 단계(HITL #2)로 진행
- **score < 80**: 불합격. retry loop 재실행. 이 채점 결과가 다음 회차의 피드백으로 전달됨
